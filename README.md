# Tester le projet

# Backend

1. installer les modules

```sh
cd backend && npm i
```

2. copier le fichier .env.example en un .env

```sh
cp .env.example .env
```

Puis entrez les valeurs souhaitées.

3. lancer le backend

```sh
npm run dev
```

Le serveur sera lancé sur le port 4000 ==> http://localhost:4000
